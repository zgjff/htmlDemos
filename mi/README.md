# Mi

这是仿照小米商城写的练习Demo。 你可以在这里查看 [小米商城官网](https://www.mi.com/)

## 运行

- 需要安装`angular`框架: `sudo npm install -g @angular/cli@latest`
- 在终端中执行`ng serve --open`可以查看此demo的最终效果.
